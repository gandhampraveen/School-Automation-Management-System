# School Management System

A comprehensive School Management System built with React.js, featuring separate dashboards for students and teachers.

## 🚀 Features

### For Students
- 📊 **Dashboard**: View attendance, marks, and assignment status
- 📅 **Attendance Tracking**: View attendance records with status badges
- 📈 **Marks & Grades**: View subject-wise marks and grades
- 📝 **Assignments**: Track pending and completed assignments
- 🕐 **Timetable**: View weekly class schedule
- 👤 **Profile**: View personal information and statistics

### For Teachers
- 👨‍🏫 **Dashboard**: Overview of class performance
- 👥 **Student Management**: View and manage students in class
- ✅ **Attendance Management**: Mark attendance for students
- 📊 **Marks Management**: Add and manage student marks
- 📋 **Reports**: Generate class reports and statistics

### General
- 🔐 **Secure Login**: Role-based authentication
- 📱 **Responsive Design**: Works on all devices
- 📊 **Data Visualization**: Charts and statistics
- 💾 **Local Storage**: Data persistence using localStorage

## 🛠️ Technology Stack

- **Frontend**: React.js
- **Styling**: CSS3 with Flexbox/Grid
- **Charts**: Chart.js
- **Icons**: Font Awesome
- **Database**: localStorage (simulated)
- **Routing**: React Router v6
